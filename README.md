# Malicious-APK
  Generate & Embed & Create Backdoor APK for Android Application

## Dependencies
  - Metasploit-Framework
  - Xterm
  - Zenity
  - Apktool
  - Zipalign

## Installation

### Download
```bash
git clone https://github.com/bsynn/Malicious-APK.git
```
### Set permission
```bash
cd Malicious-APK
chmod +x malicious_apk.sh
```
### Run
```bash
./malicious_apk.sh
```
