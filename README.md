# Malicious-APK

## Dependencies
  - Metasploit-Framework
	- Xterm
	- Zenity
	- Apktool
	- Zipalign

## Installation

### Download
```bash
git clone https://github.com/bsynn/Malicious-APK.git
```
### Setup permission
```bash
cd Malicious-APK
chmod +x malicious_apk.sh
```
## Run
```bash
./malicious_apk.sh
```
